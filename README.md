# Feral Hog Content

Website for Gabriel J. Perez and Feral Hog Content, Inc.

Live: https://feralhogcontent.com

Hosted on GitHub Pages. To update: edit `index.html` (or any file), commit, push.
GitHub Pages auto-deploys in about 1 minute.

Custom domain configured via the `CNAME` file at the repo root and DNS records at the registrar (Squarespace Domains).
